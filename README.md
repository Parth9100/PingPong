# Ping Pong Game (Raylib + C++)

A simple Pong-style game built with [raylib](https://www.raylib.com/) and C++. Move your paddle up and down to bounce the ball and score points against the CPU!

## Features

- Modern color palette (teal, orange, cream)
- Smooth paddle and ball movement
- Dotted center line
- Score display for both players
- CPU opponent

## Controls

- **Up Arrow**: Move paddle up
- **Down Arrow**: Move paddle down

## Requirements

- [raylib 5.x](https://www.raylib.com/)
- C++17 or newer
- Windows (tested with Visual Studio)

## How to Build

1. **Install raylib**  
   Download and set up raylib for your compiler ([instructions](https://github.com/raysan5/raylib/wiki/Working-on-Windows)).
2. **Open the Solution**  
   Open `HelloWorld.sln` in Visual Studio.
3. **Build and Run**  
   Press `F5` or click "Local Windows Debugger" to build and run.
